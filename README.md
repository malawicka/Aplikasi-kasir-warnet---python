# Aplikasi-kasir-warnet---python
Project ini merupakan aplikasi kasir warnet berbasis Python, dijalankan melalui terminal / PowerShell. Fitur utama:  Login dengan username &amp; password Pilih tipe pelanggan: Tetap atau Umum Input lama jam main Hitung total bayar dengan diskon otomatis untuk lama main ≥ 5 jam

Fungsi utama dari aplikasi ini adalah:

Mengelola pembayaran pelanggan berdasarkan tipe pelanggan dan lama jam main.
Memberikan diskon otomatis untuk pelanggan yang bermain lebih dari 5 jam.

Fitur utama:

Login menggunakan username & password (default: admin / admin1234)
Pilih tipe pelanggan: Tetap atau Umum
Input lama jam main
Menghitung total pembayaran secara otomatis
Diskon otomatis untuk pelanggan lama (≥ 5 jam)
